Ayuda

Esta aplicación permite la gestión de una libreta de notas, permitiendo la creación, edición y eliminación de notas.

Esta aplicación ha sido desarrollada como parte de un Trabajo de Fin de Máster del Máster Universitario en Desarrollo del Software para la Web de la Universidad de Alcalá.

Formato en las notas

El contenido de las notas soporta un formato en Markdown básico, soportando los siguientes elementos (salvo la lista):
  • Los seis niveles de encabezado, desde # (encabezado de nivel 1) hasta ###### (encabezado de nivel 6).
  • Negrita, para la cual es necesario encerrar el texto a dar formato entre **.
  • Cursiva, para la cual es necesario encerrar el texto a formatear entre *.
  • Formato de código, el cual imprimirá el texto formateado en una fuente monoespaciada. Para dar formato de código, se encierra el texto entre caracteres `.

Para separar el texto en párrafos, es necesario escribir un salto de línea o más entre ambos.

Ejemplo

# Título de la nota

Esto es un *ejemplo* de nota, en el **Markdown** soportado por la aplicación.

## Características soportadas

**Negrita**, *cursiva* y `código`.
